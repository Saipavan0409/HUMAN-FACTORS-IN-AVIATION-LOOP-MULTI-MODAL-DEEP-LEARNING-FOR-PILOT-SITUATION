# HUMAN-FACTORS-IN-AVIATION-LOOP-MULTI-MODAL-DEEP-LEARNING-FOR-PILOT-SITUATION
Pilot Situation Awareness (SA) is vital for safe flight. This work proposes a multimodal deep learning model using eye-gaze (CNN) and control data (LSTM), fused via attention, to classify SA levels. The approach outperforms single-modality methods for real-time monitoring.
